# PROJECT NVISIA  
**North Korea Visual Insight with SIA**

## 📌 프로젝트 개요

| 항목 | 상세 내용 |
| :--- | :--- |
| **프로젝트 명** | North Korea Visual Insight with SIA (NVISIA) |
| **배경 및 필요성** | 다양한 채널에서 쏟아지는 북한 관련 정보를 통합적으로 모니터링하고 분석할 수 있는 체계적인 도구의 필요성이 증대됨. |
| **목표** | 북한 동향 파악에 필요한 핵심 뉴스 데이터를 실시간으로 수집하고, 이를 직관적인 대시보드 형태로 제공하여 사용자의 의사결정을 지원함. |
| **주요 내용** | North Korea Visual Insight(NVI) 기반의 데이터 뷰어 개발. 주요 뉴스의 키워드 트렌드, 이슈별 동향 변화 등을 시각화하여 제공. |
| **기대 효과** | 북한 관련 정보 수집 및 분석에 소요되는 시간과 비용을 절감하고, 객관적이고 신속한 상황 판단을 가능하게 함. |
